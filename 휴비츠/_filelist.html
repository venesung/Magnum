이미지명 규칙 및 경로

  1. 이미지는 img폴더 안에 각 메뉴별 뎁스 아래 저장한다.
  2. 공통적인 성격을 갖거나 특수한 이미지인 경우 한 폴더 안에 저장한다.
    예시1) 각 페이지 타이틀 이미지 : img/title/title_파일명.gif
    예시2) 버튼 이미지 : img/button/btn_파일명.gif
  3. 그 외 이미지명 규칙
    1. on/off 이미지 : img/이미지명_on/off.gif
    2. 배경 이미지 : img/bg_이미지명.gif
    3. 불릿 이미지 : img/bu_이미지명.gif
    4. 비쥬얼/그래프등 기타 이미지 : img/img_이미지명.jpg
    5. 사진 이미지 : img/pic_이미지명.jpg
    6. 아이콘 이미지 : img/icon_이미지명.gif
    7. 텍스트 이미지 : img/text_이미지명.gif
    8. 2Depth 타이틀 이미지 : img/stitle_이미지명.gif
    9. 3DEpth 타이틀 이미지 : img/sstitle_이미지명.gif
    10. 탭 이미지 : img/tab_이미지명.gif

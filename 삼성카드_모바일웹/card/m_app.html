앱카드란?

회원님께서 보유하고 계신 실물카드(신용/체크) 정보를 APP에 등록 후, 온라인/일반가맹점에서 편리하게 이용하실 수 있는 모바일 결제서비스입니다.

앱카드 등록 가능 회원

  • 삼성 개인신용, 체크, 법인개별카드를 보유한 회원님(가족카드 등록 불가)
  • 본인/개인 명의 휴대폰 이용 회원님(SKT, KT, LG U+,SKT MVNO(알뜰폰))
  • 스마트폰(iOS, Android) 이용 회원님
m포켓 다운로드
삼성 앱카드 등록 방법

앱카드는 휴대폰 본인인증과 신용카드(혹은 공인인증서) 인증을 통해 등록 가능합니다.

  1. m포켓 다운로드
    구글play스토어 혹은 앱스토어에서 m포켓 앱 다운로드
  2. 서비스 등록
    휴대폰, 카드정보, 공인인증서 중 하나의 인증방법을 선택하여 본인인증 후 서비스 등록
  3. 앱카드 등록
    카드정보 혹은 공인인증서를 통해 카드정보 인증 후 앱카드 등록
  4. 결제 비밀번호 6자리 설정 및 서명 입력
이용방법

- 온라인/모바일 쇼핑몰 이용 시

  1. 온라인 쇼핑몰 결제 창에서 "앱카드로 결제" 선택
  2. m포켓 실행 후 결제코드입력
  3. 결제정보 확인 후 앱카드 비밀번호 입력
  4. 내역 확인 후 m포켓 종료
  5. PC화면에서 결제코드 확인 후 종료

- 일반 가맹점에서 이용 시

  1. m포켓 실행
  2. 결제하고자 하는 앱카드를 선택
  3. 설정에서 앱카드 결제 방식 선택 (바코드, QR코드, NFC)
  4. 앱카드 결제 선택 후, 결제비밀번호 입력
  5. 매장 직원에게 결제 요청 및 결제 완료
이용가능 가맹점
  1. 온라인/모바일 쇼핑몰
    - 삼성카드 결제가 가능한 모든 온라인/모바일 쇼핑몰에서 사용 가능
  2. 오프라인 가맹점
    - 대형마트 : 하나로클럽 전점, 홈플러스(4/17~), 롯데마트(4/18~)
    - 편의점 : 세븐일레븐/바이더웨이 전점
    - 롭스 전점(헬스 앤 뷰티스토어)
    - 명동 앱카드 Zone : 명동지역 140여 개 가맹점
     (PC 홈페이지 內 가맹점 자세히 보기 확인)
기타유의사항
  • 개인 신용/체크카드 및 법인 개별카드 이용 시, 카드 종류에 관계없이 앱 카드를 등록하실 수 있습니다.
  • 법인공용/선불/올앳/기프트/가족카드로는 이용하실 수 없습니다.
  • NFC를 통해 결제를 시도하실 경우, 앱카드가 실행 중이어야 합니다.

[ 규칙 및 경로 ]

  1. 모든 이미지는 img/common 폴더 안에 저장한다.
    1. 이미지 경로 : img/common/이미지명.gif
  2. 이미지명 규칙
    1. on/off 이미지 : 이미지명off/on.gif
    2. 배경 이미지 : bg_이미지명.gif
    3. 블릿 이미지 : bl_이미지명.gif
    4. 비쥬얼/그래프등 기타 이미지 : img_이미지명.jpg
    5. 사진 이미지 : pic_이미지명.jpg
    6. 아이콘 이미지 : icon_이미지명.gif
    7. 텍스트 이미지 : txt_이미지명.gif
    8. 타이틀 이미지 : tit_이미지명.gif
    9. 탭 이미지 : tab_이미지명.gif
  3. 그 외 규칙
    1. 주석 : <!-- list --> <!-- // list -->
    2. 완료 : <!-- 0101 완료 --><!--// 0101 완료 -->
    3. clear : <div class="clear"><!-- clearBlock --></div>

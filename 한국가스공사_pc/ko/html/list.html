[ 규칙 및 경로 ]

  1. 이미지는 img폴더 안에 각 메뉴별 뎁스 아래 저장한다.
  2. 공통적인 성격을 갖거나 특수한 이미지인 경우 한 폴더 안에 저장한다.
    1. 타이틀: images/tit/tit_이미지명.gif
    2. 버튼: images/btn/btn_이미지명.gif
    3. 탭: images/tab/tab_이미지명.gif
    4. 아이콘/블릿/라인: images/common/이미지명.gif
    5. 텍스트: images/서브폴더/txt_이미지명.gif
    6. 배경: images/서브폴더/bg_이미지명.gif
    7. 기타: images/서브폴더/관련내용_이미지명.gif
  3. 이미지명 규칙
    1. on/off 이미지 : 이미지명off/on.gif
    2. 배경 이미지 : bg_이미지명.gif
    3. 블릿 이미지 : bl_이미지명.gif
    4. 비쥬얼/그래프등 기타 이미지 : img_이미지명.jpg
    5. 사진 이미지 : pic_이미지명.jpg
    6. 아이콘 이미지 : icon_이미지명.gif
    7. 텍스트 이미지 : txt_이미지명.gif
    8. 타이틀 이미지 : tit_이미지명.gif
    9. 탭 이미지 : tab_이미지명.gif
  4. 그 외 규칙
    1. 주석 : <!-- list --> <!-- // list -->
    2. 완료 : <!-- 0101 완료 --><!--// 0101 완료 -->
    3. clear : <div class="clear"><!-- clearBlock --></div>
  5. 페이지 폴더명
    1. 메인 : <main>
    2. KOAGS : < intro >
    3. 사업소개 : < business >
    4. 경영정보 : < manageinfo >
    5. 가스정보 : < gasinfo >
    6. KOGAS 홍보 : < promotion >
    7. KOGAS 서비스 : < service >
    8. 통합검색 : < search >
    9. 정보공개 : < release >
    10. 경영공시 : < disclosure >

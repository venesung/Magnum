전월 이월금액 : 전월 초과 이용금액 / 당월 이용금액 : 당월에만 이용한 금액
월 이용금액 : 전월 초과 이용금액 + 당월 이용금액 / 미달 이용금액 : 월 약정구간 최소금액 - 월 이용금액
초과 이용금액 : 월 이용금액 - 월 약정구간 최대금액 / 누계 이용금액 : 약정기간동안 총 이용한 금액

  1. 월 약정구간 미만으로 이용한경우, 미달이용금액(월 약정구간 최소금액- 대상카드 이용금액 )× 2%가  약정한도에서 선 차감되며,
    부족시 카드대금으로 청구 됩니다.
    단, 월별 차감 및 청구 합계금액은 (약정시 부여한 약정한도 ÷ 약정기간의 총개월수)를 초과하지 않습니다.

  2. 월 약정구간 최대금액을 초과하여 이용하면 초과 이용금액은 다음달로 이월됩니다.
    (단, 약정변경 및 종료 시에는 이월되지 않음)

  3. 약정기간 중 수퍼S카드 이용금액은 약정기간동안 누계로 합산되어, 월말에 누계이용금액이 조기종료대상액 이상인 경우에는
    약정이 조기종료되며 다음달에 동일한 약정으로 약정한도가 재부여 됩니다.

  4. 이용금액은 수퍼S카드 본인 및 가족카드의 합계금액 입니다.

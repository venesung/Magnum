삼성카드에서 적립한 CJ ONE 누적 포인트 기준입니다.
CJ ONE 적립 및 사용 상세 확인은 CJONE 홈페이지에서 확인할 수 있습니다.

* CJ ONE 누적포인트란 CJ ONE 삼성카드 이용 실적 기준으로, 삼성카드에서 적립한 CJONE 포인트를 의미합니다.

매출 적립포인트

  1. 적립대상 : 전월 일시불, 할부 이용금액이 20만원 이상 회원

  2. 적립매출 : CJ ONE 멤버십 대상 가맹점 이용시

  3. 적립율 : 0.6 ~ 10% (CJ ONE 멤버십 가맹점별로 상이하며, 세부 내용은 CJ ONE 삼성카드에서 확인가능합니다.)

특별 적립포인트

  1. 적립대상 : 전월 일시불, 할부 이용금액이 20만원 이상 사용한 회원

  2. 적립매출 : CJ ONE 멤버십 가맹점 외에서 사용하신 금액
    적립 제외 매출 : CJ ONE 멤버십 가맹점, 무이자할부, 국세/지방세, 공과금, 대중교통이용액, 산재/고용보험료, 건강보험료,
         올앳 카드 충전금액, S-infocare, 관리비 info서비스

  3. 적립율 : 0.5%

  4. 적립시점 : 전월 이용하신 카드 사용액에 대하여 당월 20일 경에 적립처리됩니다.
                   예) `11년 7월 1일 ~ 31일까지 이용한 카드 사용액 기준 8월 20일경에 적립

  1. CJ ONE 가용포인트는 삼성카드에서 적립한 포인트, CJ ONE 멤버십 적립 포인트, CJ ONE 포인트
    사용내역을 모두 포함된 것입니다.

  2. 상기 CJ ONE 가용포인트는 매월 5일 기준입니다. 현재 시점 회원님의 CJ ONE 가맹점에서 사용하실 수
    있는 가용포인트와 내용이 다를 수 도 있습니다.

  3. CJ ONE 포인트의 소유/운영권은 모두 CJ ONE에 귀속되어 있으며, 포인트 사용/소멸관련 등 기타 궁금사항이
    있을 경우 CJ ONE으로 문의 바랍니다. (홈페이지 : www.cjone.com / 전화번호 : 1577-8888)

  4. CJ ONE 포인트의 유효기간은 24개월입니다.

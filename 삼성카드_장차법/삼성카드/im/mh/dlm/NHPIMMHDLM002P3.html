사본 문서는 소득공제용 사용금액 확인서의 인터넷(홈페이지/이메일)발급시 적용되는 위변조 방지 솔루션에 대한 장애조치 사항에 대하여 작성한 것입니다.

프린트 창에서 지원불가 프린트라고 나타나는 경우

  • 소득공제 확인서 원본 출력을 위해서는 해당 프린트에 맞는 정확한 드라이버를 설치하셔야 합니다.
  • 마크프린터나 워터마킹 등 보안설정을 위한 드라이버가 설치되어 있으면 지원이 안되므로 삭제해야만 합니다.
  • 프린트 확인은 시작버튼 >설정 >프린트 에서 확인하실 수 있습니다.

수동설치 프로그램을 클릭하여 ActiveX(위변조방지 클라이언트 모듈)를 수동으로 설치하시기 바랍니다.

수동설치 프로그램 다운로드

인터넷으로 법적 효력이 있는 문서를 발급하기 위해서는 해당 출력 문서에 적합한 보안장치를 출력하기 위한 테스트를 필요로 합니다.
이들 보안장치를 출력테스트를 통과한 프린터가 발급가능 프린터로 분류되게 됩니다.

지원가능 프린터 목록보기

  • 전환취소는 신청한 당일 21시(오후 9시)까지만 가능합니다.

발급 가능한 프린터 중 프린터의 연결방식 (로컬, 윈도우 공유, IP 네트워크)에 따라 발급불가 프린터가 구분됩니다.
로컬(PC에 직접 연결) 프린터와 IP주소를 이용한 네트워크 프린터는 지원되며, 타 PC에 연결된 프린터를 윈도우에서 공유하여 사용하는 방식의 프린터는 보안 위험성으로 인하여 지원하지 않습니다.
(출력데이터가 PC에 남음) 그리고 프린터의 실체가 없고 출력물을 컴퓨터 상의 이미지 파일로 출력시키는 가상프린터(예 PDFwriter나 FinePrint 등) 역시 발급불가 프린터 입니다.

화면 캡쳐 프로그램을 사용 중 혹은 사용자의 화면을 타인에게 전송할 수 있는 기능을 가진 프로그램
(예, 화면캡쳐 프로그램)이 실행 중에는 이를 자동으로 감지하고 메시지를 출력한 후, 화면보호 기능 동작하여 증명서의 조회 및 출력화면을 회색으로 가립니다.
해당 프로그램의 실행여부를 확인 후 종료해야 하며, 종료 시 조회화면은 바로 정상으로 돌아옵니다.

인터넷 증명서의 도장/프레임은 이미지(그림) 파일로 구성되어 있습니다. 따라서 인터넷 통신 속도에 영향으로
이미지 파일이 로딩되지 이전에 증명서를 출력한 경우, 이미지 부분만 출력이 안될 수 있습니다.
화면 상에 증명서가 완전한 형태로 나타나 이후 출력 버튼을 누르십시오

서비스팩2의 ‘팝업차단’ 기능으로 인해 인터넷 증명서 창이 정상적으로 띄어지지 않을 수 있습니다. 아래와 같이 설치를 진행하여 주시기 바랍니다.

  • 해당 사이트에 접속한 상태에서 팝업차단 메시지가 주소 창 아래에 나올 경우, 상단의 상태 줄을 클릭하신 후
    현재 사이트의 팝업을 항상 허용(A)을 선택 하시면 됩니다.
  • 주소 표시줄 아래 ‘제공처에서 배포한 Active X를 설치하려면 여기를 클릭하십시오’ 라는 부분을 클릭 하신 후,
    Active X 컨트롤 설치(C) 를 선택하시기 바랍니다.
  • 이 소프트웨어를 설치하시겠습니까? 라는 보안경고창이 뜨면, 여기에서 설치(I)를 선택하시면 됩니다.
    설치 후 웹 페이지가 새로고침 되면 기능이 정상적으로 동작하게 됩니다.
